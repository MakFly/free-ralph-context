# Environment Variables

Configure Ralph via environment variables.

## Configuration File

Create `~/.ralph/config.json`:

```json
{
  "server": {
    "port": 3847,
    "host": "localhost"
  },
  "claudeCode": {
    "sessionsPath": "~/Library/Application Support/Claude/claude_desktop_config/sessions",
    "transcriptsPath": "~/Library/Application Support/Claude/claude_desktop_config/transcripts"
  },
  "features": {
    "autoCompress": true,
    "compressThreshold": 0.7,
    "checkpointInterval": 300
  }
}
```

## Variables

### Server

```bash
RALPH_PORT=3847
RALPH_HOST=localhost
RALPH_DEV_MODE=false
```

### Claude Code Paths

```bash
# macOS
RALPH_SESSIONS_PATH="~/Library/Application Support/Claude/claude_desktop_config/sessions"
RALPH_TRANSCRIPTS_PATH="~/Library/Application Support/Claude/claude_desktop_config/transcripts"

# Linux
RALPH_SESSIONS_PATH="~/.config/Claude/claude_desktop_config/sessions"
RALPH_TRANSCRIPTS_PATH="~/.config/Claude/claude_desktop_config/transcripts"

# Windows
RALPH_SESSIONS_PATH="%APPDATA%\\Claude\\claude_desktop_config\\sessions"
RALPH_TRANSCRIPTS_PATH="%APPDATA%\\Claude\\claude_desktop_config\\transcripts"
```

### Features

```bash
# Auto-compression
RALPH_AUTO_COMPRESS=true
RALPH_COMPRESS_THRESHOLD=0.7  # 70%

# Checkpoints
RALPH_CHECKPOINT_INTERVAL=300  # seconds

# Memory
RALPH_MEMORY_ENABLED=true
RALPH_MEMORY_MAX_ITEMS=1000

# Metrics
RALPH_METRICS_ENABLED=true
RALPH_METRICS_RETENTION_DAYS=90
```

### Debugging

```bash
RALPH_DEBUG=true
RALPH_LOG_LEVEL=info  # debug, info, warn, error
RALPH_LOG_FILE=~/.ralph/ralph.log
```

## Example .env File

```bash
# Server
RALPH_PORT=3847
RALPH_HOST=localhost

# Claude Code
RALPH_SESSIONS_PATH="~/Library/Application Support/Claude/claude_desktop_config/sessions"
RALPH_TRANSCRIPTS_PATH="~/Library/Application Support/Claude/claude_desktop_config/transcripts"

# Features
RALPH_AUTO_COMPRESS=true
RALPH_COMPRESS_THRESHOLD=0.7
RALPH_CHECKPOINT_INTERVAL=300

# Debug
RALPH_DEBUG=false
RALPH_LOG_LEVEL=info
```

## Priority

Configuration is loaded in this order:

1. Environment variables (highest priority)
2. `~/.ralph/config.json`
3. Default values (lowest priority)

> **Tip:**
> **Use .env files**: For local development, create a `.env` file in your project root.

# Installation

Get Ralph Dashboard up and running on your system.

## Prerequisites

- **Node.js** 18+ and npm/yarn/bun
- **Claude Code** latest version
- **MCP Server** enabled in Claude Code

## Install via Package Manager

```bash
# Using npm
npm install -g @ralph/dashboard

# Using yarn
yarn global add @ralph/dashboard

# Using bun
bun install -g @ralph/dashboard
```

## Manual Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/ralph-dashboard.git
cd ralph-dashboard

# Install dependencies
bun install

# Start the dashboard
bun run dev
```

## Configuration

Create a `~/.ralph/config.json` file:

```json
{
  "server": {
    "port": 3847,
    "host": "localhost"
  },
  "claudeCode": {
    "sessionsPath": "~/Library/Application Support/Claude/claude_desktop_config/sessions",
    "transcriptsPath": "~/Library/Application Support/Claude/claude_desktop_config/transcripts"
  },
  "features": {
    "autoCompress": true,
    "compressThreshold": 0.7,
    "checkpointInterval": 300
  }
}
```

## Verify Installation

```bash
ralph --version
ralph dashboard start
```

Visit `http://localhost:3847` to see your dashboard.

> **Success:**
> **Installation Complete!** You should now see the Ralph Dashboard running.

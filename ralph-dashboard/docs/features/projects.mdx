# Project Management

Manage and organize your Claude Code projects.

## Project List

View all projects on the **Projects** page (`/projects`).

### Project Information

Each project shows:

- **Name**: Project folder name
- **Path**: Full file system path
- **Context Usage**: Visual progress bar
- **Status**: Current health indicator
- **Last Activity**: When last updated
- **Actions**: Quick action menu

## Project Details

Click any project to see detailed information:

### Context Gauge

Large circular gauge showing:

- Percentage used
- Current tokens
- Maximum tokens (200k)
- Color-coded status

### Stats Grid

| Stat          | Description           |
| ------------- | --------------------- |
| Tokens Used   | Current token count   |
| Context Usage | Percentage of max     |
| Last Updated  | Last activity time    |
| Status        | Current health status |

### Quick Actions

- **Create Checkpoint**: Save current state
- **Compress Context**: Reduce token usage
- **Spawn New Session**: Fresh context
- **Clear Session**: Delete session data

## Project Actions

### View Transcript

See the raw conversation transcript:

```typescript
// Opens transcript file in default editor
ralph_open_transcript(projectName)
```

### Clear Session

Remove project from dashboard:

```typescript
ralph_clear_session(projectName)
```

> **Warning:**
> **Clearing is permanent!** Make sure to create a checkpoint first if you need the data.

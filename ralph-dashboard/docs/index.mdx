## Getting Started

1. **Install Ralph MCP**: `npm install @ralph/mcp`
2. **Configure Claude Code**: Add Ralph to your Claude Code config
3. **Start the Dashboard**: `ralph-dashboard start`

> **Tip:**
> **Pro Tip**: Enable automatic compression to keep your sessions optimized without manual intervention.

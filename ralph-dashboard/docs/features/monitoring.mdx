# Real-time Monitoring

Monitor all your Claude Code projects in real-time.

## Dashboard Overview

The main dashboard provides:

### Stats Cards

- **Active Projects**: Number of tracked sessions
- **Total Tokens**: Combined context usage
- **Critical**: Projects needing immediate attention
- **Warning**: Projects approaching limits

### Project Table

| Column  | Description               |
| ------- | ------------------------- |
| Project | Name and path             |
| Context | Usage bar with percentage |
| Tokens  | Current / Maximum tokens  |
| Status  | Health indicator badge    |
| Updated | Last activity timestamp   |
| Actions | Quick link to details     |

## Live Updates

The dashboard updates every **500ms** via Server-Sent Events (SSE):

```typescript
// SSE connection is automatic
const { status, lastEvent } = useRalphSSE()

// status.projects updates in real-time
// lastEvent shows latest event type
```

### Event Types

- `init`: Initial data load
- `update`: Project state change
- `sync:progress`: File sync progress
- `sync:error`: Sync error occurred

## Connection Status

The header shows connection status:

- **Live**: Connected and receiving updates
- **Connecting**: Attempting to connect
- **Error**: Connection failed (retrying automatically)

> **Tip:**
> **Auto-refresh**: The dashboard automatically reconnects if the connection drops.

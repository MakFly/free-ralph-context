# Metrics Dashboard

Track your development metrics over time.

## Available Metrics

### Token Usage

Track token consumption:

- Total tokens used
- Tokens per session
- Average growth rate
- Compression savings

### Session Activity

Monitor session patterns:

- Active sessions per day
- Session duration
- Most active projects
- Peak usage times

### Memory Operations

Memory system stats:

- Memories created
- Memory access frequency
- Categories distribution
- Search success rate

## Visualizations

### Timeline Chart

View metrics over time:

```typescript
const chartData = {
  labels: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri'],
  datasets: [
    {
      label: 'Tokens Used',
      data: [45000, 52000, 48000, 61000, 55000],
    },
  ],
}
```

### Distribution Chart

See category breakdown:

- Memory types
- Project distribution
- Status distribution

### Heat Map

Activity patterns:

- Hour of day
- Day of week
- Project activity

## Exporting Data

Export metrics for analysis:

```typescript
// Export as JSON
ralph_export_metrics({ format: 'json' })

// Export as CSV
ralph_export_metrics({ format: 'csv' })

// Custom date range
ralph_export_metrics({
  start: '2024-01-01',
  end: '2024-01-31',
})
```

> **Info:**
> **Metrics retention**: By default, metrics are kept for 90 days. Configure this in settings.

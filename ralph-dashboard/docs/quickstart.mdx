# Quick Start

Get started with Ralph Dashboard in 5 minutes.

## 1. Start the Dashboard

```bash
ralph dashboard start
```

The dashboard will open at `http://localhost:3847`.

## 2. Open Claude Code

Create or open a project in Claude Code. Ralph will automatically detect it.

## 3. Monitor Your Session

On the dashboard, you'll see:

- **Active Projects**: List of all Claude Code sessions
- **Context Usage**: Real-time token consumption
- **Status Indicators**: Health status for each project

## 4. Create a Checkpoint

When you reach an important milestone:

```typescript
// In Claude Code, use the MCP tool
ralph_checkpoint('Feature complete - ready for testing')
```

## 5. Compress Context

When context usage is high:

```typescript
// Automatically compress with AI
ralph_compress()
```

## Common Workflows

### Starting a New Feature

```typescript
1. ralph_checkpoint("Before starting new feature")
2. Work on your feature
3. ralph_compress() // if context gets high
4. ralph_checkpoint("Feature implemented")
```

### Debugging Session

```typescript
1. ralph_checkpoint("Before debugging")
2. Debug and fix issues
3. ralph_compress() // clean up debug logs
4. ralph_checkpoint("Bug fixed")
```

> **Info:**
> **Keyboard Shortcut**: Press `Cmd/Ctrl + R` in the dashboard to force refresh all projects.

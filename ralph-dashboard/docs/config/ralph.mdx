# Ralph Settings

Configure Ralph MCP behavior.

## Settings Location

`~/.ralph/mcp.json`

## Compression Settings

```json
{
  "compression": {
    "enabled": true,
    "targetRatio": 0.25,
    "preserveRecent": 50,
    "strategy": "smart"
  }
}
```

### Strategies

- `smart`: AI-selected content to preserve
- `recent`: Keep only recent messages
- `important`: Keep high-priority content
- `aggressive`: Maximum compression

## Memory Settings

```json
{
  "memory": {
    "enabled": true,
    "maxItems": 1000,
    "autoCapture": {
      "enabled": true,
      "categories": ["decision", "error"],
      "minPriority": "normal"
    },
    "retention": {
      "days": 90,
      "maxAccessCount": 10
    }
  }
}
```

## Checkpoint Settings

```json
{
  "checkpoint": {
    "autoCreate": {
      "enabled": false,
      "triggers": ["context>80", "error", "milestone"]
    },
    "compression": {
      "enabled": true,
      "level": "medium"
    },
    "retention": {
      "maxCount": 50,
      "maxAgeDays": 30
    }
  }
}
```

## Spawn Settings

```json
{
  "spawn": {
    "autoSuggest": true,
    "threshold": 0.85,
    "defaultOptions": {
      "keepMemories": true,
      "clearHistory": true
    }
  }
}
```
